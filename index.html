About ClippedAI

ClippedAI is a free, open-source tool that automatically transforms long-form videos into viral YouTube Shorts. Unlike cloud-based tools, ClippedAI runs locally on your machine, ensuring your content and data remain private. It leverages advanced AI models for transcription, smart clip detection, and subtitle generation, and is built on the robust clipsai library.

Key Benefits

  • 100% free and open-source (no subscriptions, no hidden fees)
  • Works offline for maximum privacy
  • Highly customizable for developers and creators
  • Supports batch processing and transcription caching
  • Perfect for podcasts, interviews, lectures, and more

üöÄ Quick Setup Guide

Setting up ClippedAI is easy! All you need is Python, FFmpeg, and free API keys from HuggingFace and Groq. Follow these steps to get started in minutes. For advanced configuration, see the full documentation.

1

Clone Repository

git clone https://github.com/Shaarav4795/ClippedAI.git
cd ClippedAI
2

Create Virtual Environment

python3 -m venv env
source env/bin/activate
3

Install Dependencies

pip install -r requirements.txt
4

Install FFmpeg

# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt install ffmpeg
5

Configure API Keys

Get free API keys from HuggingFace and Groq, then update main.py

View API Setup Guide

üîë API Keys Setup (100% Free)

HuggingFace Token

  1. Sign up at HuggingFace
  2. Request access to Pyannote repositories
  3. Create API token with "read" permissions
  4. Replace placeholder in main.py

Groq API Key

  1. Sign up at Groq
  2. Get API key from dashboard
  3. Replace placeholder in main.py
  4. Start generating viral titles!
